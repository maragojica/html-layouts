File Management

Add documents, spreadsheets presentations, videos, images, PDFs, and more in the Files tile within your project workspace. Select the Files tab in the Discussion tile to view all the files in your project. These files will be available as you and your team transition from milestone to milestone in any given project. Select the folder you would like to add a file or folder in and click the New button to create a new folder or file. You can select one or more files and delete them from a given folder within the tile.

To delete files, select the files you would like to delete, and then select the Delete button.

Select the Trash Bin in your file menu to view all deleted files. To restore files previously deleted, select the files you would like to restore and select the Restore button to confirm the renewal of your files.
