Course Administration

Course Administration through BlendED™ allows users to add new courses and categories, import courses, restore and backup courses, and assign course access levels. Assigning instructors and students to courses and managing classroom access is made simple. Administrators can configure, duplicate, and manage course shells for effortless management of course sections. With a Learning Management System (LMS) event activity log and analytics models, courses are administered with institutional effectiveness in mind.

Manage Course Categories

Use Course Categories to organize courses within your institution's Learning Management System with BlendED™.

Click on Settings in the Left Navigation menu to access BlendED™ Settings.

Go to the Blended™ Settings tile and select the VIEW MORE button.

Click on the Courses tab. Then, select the Manage Courses and Categories hyperlink in the Courses section.

Go to Course Categories and select the CREATE NEW CATEGORY button to create a new category.

Input as much information as possible in each field.

Click on the CREATE CATEGORY button to complete the setup of your new category.

The new category will now appear under Course Categories.

Select the Settings icon on any given category to modify its settings.
