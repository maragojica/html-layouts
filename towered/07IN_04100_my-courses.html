My Courses

The My Courses screen allows BlendED™ users to view all active courses in one centralized location.

To navigate to My Courses, click on My Courses on the Left Navigation menu.

Click on the CUSTOMIZE THIS PAGE button to view the My Courses screen in edit mode. Within this mode, you can modify the settings of displayed blocks, and add additional blocks to enhance your BlendED™ experience.

Click on the ADD BLOCK button to display the Add a Block menu.

Remember, the Add a Block screen only displays blocks that are not already on the front page.

You can move the position of blocks by clicking and dragging the move icon.

Click on the STOP CUSTOMIZING THIS PAGE button when done making changes.
