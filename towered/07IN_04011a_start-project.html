Project Planning

Our collaborative platform is designed to guide you through Course and Certificate Development Projects. Create projects from your interests on the profile screen or start a new project on the My Projects screen. While in the project space, complete tasks to advance to milestones. Invite additional partners while you work. Add files and send messages to partners for effective collaboration. Stay tuned as we expand our project offerings to peer reviews, study abroad programs, and more.

Start Project

Educators can create and manage projects with ease using CollaboratED™. Select My Projects from the Left Navigation menu to access your projects. Then, select the New Project button to enter information about the new project you would like to create.

Complete relevant information about your project such as the project type, name, description, subject area, and start and end dates. Select the Create button to develop your project and to begin collaborating with your peers.
